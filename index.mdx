C++ Baremetal Senders and Receivers is a header-only library that brings structured, composable async programming to microcontrollers. It implements a subset of [P2300](https://wg21.link/p2300) — the proposed C++ standard for asynchronous execution — tuned for environments without dynamic memory allocation or exceptions.

## What you can build

This library gives you the building blocks to express async firmware flows declaratively — scheduling work on priority interrupts, waiting for timer events, handling message responses, and managing cancellation — without threads or an RTOS.

## Key features

- **Zero allocation** — no heap; operation states use static or stack allocation
- **No exceptions** — errors flow through a dedicated error channel as values
- **Interrupt-driven** — `fixed_priority_scheduler` and `time_scheduler` integrate directly with ISRs
- **Composable** — pipe syntax (`|`) chains senders for readable async flows
- **Cancellable** — cooperative cancellation via `inplace_stop_source` / `inplace_stop_token`
- **Debuggable** — compile-time debug signal framework for tracing async execution
- **Standards-aligned** — intentionally close to P2300 / `std::execution` naming
